Software Engineer

Job Description

    Participate in software development projects and implement individual application components using Java, SQL, Spring and Restful technology. Perform unit and local testing to verify the functionality of the responsible components. Support testing cycles and production by debugging code issues and implementing code fixes. Collaborate with Project Management team and cross functional teams. Learn functional domain and business logic from system documents and senior team members. Academic background in software design and development with the focus on: OOP, data structures and algorithms. Proficient in Java/J2EE, SQL, Spring and Restful services. Hands-on experience in these technologies desirable. Knowledge in relational (Oracle) and no-SQL databases. Excellent problem solving skills. Clear and effective communication skills. Motivated teammate. Self-starter with dedication to professional growth.
Preferred Qualifications
  • BS degree in Computer Science or Information Management
  • Excellent analytical and quantitative skills
  • Excellent communication and collaboration skills and ability to multi-task
  • To be successful you are self-motivated; driven to achieve and exceed expectations
